# 🚀 DigiTools – Digital Tools Buying Website

## 📌 Project Name

**DigiTools – A Digital Tools Buying Platform**

---

## 📖 Description

DigiTools is a modern and fully responsive digital tools marketplace where users can explore, compare, and purchase premium productivity tools, templates, and software. The platform provides a clean UI, smooth user experience, and interactive cart functionality to simulate a real-world e-commerce system for digital products.

---

## ⚙️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS & DaisyUI
* 🧠 JavaScript (ES6+)
* 🔔 React Toastify
* 📦 JSON (for product data handling)

---

## ✨ Key Features

### 🛒 1. Dynamic Cart System

* Add/remove products instantly
* Real-time cart count update in navbar
* “Proceed to Checkout” clears the cart completely

### 🔄 2. Toggle-Based UI System

* Switch between **Products** and **Cart** view
* Default product view with smooth interaction
* Empty cart state handling

### 📱 3. Fully Responsive Design

* Optimized for mobile, tablet, and desktop
* Clean layout based on Figma design
* Adaptive UI components for better UX

---

## 🌟 Additional Highlights

* Toast notifications for all actions (add, remove, checkout)
* Structured product data using JSON
* Interactive product cards with tags and features
* Clean and scalable component-based architecture
